# Aria-of-Arcana

Aria of Arcana is an RPG-style educational music game based on the ANYRPG framework. It combines fantasy adventure with interactive music learning, allowing players to explore a magical world while developing a foundational understanding of music theory.

Throughout the game, players encounter musical challenges, puzzles, and interactive gameplay elements designed to introduce essential concepts such as:

Pitch
Rhythm
Harmony
Musical patterns
Core music theory fundamentals

By blending immersive gameplay with educational content, Aria of Arcana transforms music theory into an engaging and intuitive experience. The game is designed especially for beginners who want to build a strong musical foundation while enjoying a story-driven adventure.

This project demonstrates the integration of game development and music education through creative digital learning experiences, using the ANYRPG system as its core framework.
